# guitarChord

吉他和弦推导算法，外带和弦指法svg可视化。

[在线试玩地址](https://youngdro.github.io/guitarChord/index.html)

核心算法及绘图位于 `src/guitarChord.js`

具体`实现过程`及`基础乐理解释`可戳[这儿](https://juejin.im/post/5b2627d051882574ac7848a4)查看

简单使用`create-react-app`搭建。

本地启动项目：
```js
// npm install
npm start
```
静态项目打包：

```js
npm run build
```
